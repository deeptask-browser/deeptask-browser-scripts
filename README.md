# DeepTask Community Scripts

A collection of community-contributed scripts for [DeepTask Browser](https://github.com/deeptask-browser/deeptask-browser). Browse by category and click **Install** to open the script directly in your DeepTask app.

## Contributing

1. Fork this repository
2. Add your script under `scripts/<category>/your-script.mjs`
3. Add a row to the relevant category table in this README
4. Open a pull request

Scripts must include metadata comments at the top — see any existing script for the format.
